# Bitgesell Snapshot

<img src="Icon.png" style="height: 60px;" />

A lightweight utility for bootstrapping a Bitgesell node using verified blockchain snapshots. Download, verify, and install the latest snapshot to reduce initial synchronization time from hours or days to just minutes.

## Getting Started

### Option 1 — Download the repository

```bash
git clone https://github.com/naftalimurgor/bitgesell-snapshots.git
cd bitgesell-snapshots

chmod +x install.sh
./install.sh
```

### Option 2 — One-line installer (Recommended)

```bash
curl -fsSL https://snapshot.bitgesell.dev/install.sh | bash
```
